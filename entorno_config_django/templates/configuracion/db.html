🔌 Soporte de Base de Datos

Django incluye de fábrica soporte para múltiples bases de datos relacionales como PostgreSQL, MySQL, SQLite y Oracle. En este proyecto estamos utilizando SQLite, que guarda toda la base de datos en un único archivo local (db.sqlite3).

El ORM nos permite interactuar con las tablas usando código Python puro, sin escribir sentencias SQL a mano. Las migraciones controlan y versionan la estructura de la base de datos.
